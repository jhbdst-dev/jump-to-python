### 02-1 숫자형 ― 로봇 제어의 기초 언어

숫자형은 단순 계산을 위한 자료형이 아니다.
로봇에서 숫자는 상태(State) 이고, 의사결정의 기준이며, 출력(Output) 이다.

로봇은 결국 다음 구조로 움직인다:

- 현재 상태 측정 → 목표와 비교 → 오차 계산 → 출력 계산 → 물리적 움직임

이 모든 과정은 정수(int), 실수(float), 그리고 사칙연산으로 이루어진다.

## 1. 숫자형의 종류와 로봇에서의 의미

# 1) 정수 (int)

정수는 카운트, 인덱스, 반복, 상태 번호를 표현한다.

예:

- 제어 루프 반복 횟수
- 모터 ID
- 스텝 번호
- 격자 좌표

step = 10
motor_id = 3

# 2) 실수 (float)

실수는 연속적인 물리량을 표현한다.
예:

- 거리 (m)
- 속도 (m/s)
- 각도 (degree, radian)
- 센서 값
- 배터리 전압

distance = 3.52
velocity = 1.75
angle = 45.3

⚠ 로봇에서는 대부분의 센서 값이 실수이다.

## 2. 로봇에서의 핵심 계산 구조

# 1) 거리 계산

distance = ((x2 - x1)**2 + (y2 - y1)**2) \*\* 0.5

의미:

- 현재 위치와 목표 위치 사이 거리 계산
- 목표 도달 여부 판단
- 충돌 회피 기준

이 계산은 자율주행의 기본 수학이다.

# 2) 속도 계산

velocity = distance / time

의미:

- 이동 속도 결정
- 모터 출력 계산
- 가속도 계산의 기초

# 3) 각도 계산 (방향 제어)

angle_error = target_angle - current_angle

의미:

- 목표 방향과 현재 방향의 차이 계산
- 로봇 회전 제어
- 자율주행 핵심 요소

# 4) 오차 계산 (제어의 시작점)

error = target - current

의미:

- 목표와 현재 상태의 차이
- 제어 알고리즘(P, PID)의 핵심 입력값
- 출력 방향 결정

⚠ 오차의 부호는 매우 중요하다.
부호가 바뀌면 로봇은 반대 방향으로 움직인다.

# 5) 비율 계산

percentage = current / max_value

활용:

- 배터리 잔량 계산
- 모터 출력 비율
- 센서 값 정규화 (0~1 스케일 변환)

# 6) 주기 제어 (% 연산자)

if step % 10 == 0:
print("10번마다 실행")

의미:

- 일정 주기로 로그 출력
- 센서 주기적 읽기
- 제어 루프 분기

제어 시스템은 항상 반복 루프로 동작한다.

## 4. 핵심 개념 정리

숫자형은 단순 계산이 아니라:

- 상태를 표현하는 도구
- 목표와의 차이를 계산하는 도구
- 출력 방향을 결정하는 도구
- 제어 시스템의 기초 수학

로봇은 결국
숫자를 입력받아 → 숫자를 계산하고 → 숫자를 출력하는 시스템이다.
