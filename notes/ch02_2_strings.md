# ch02-2 문자열 (Strings)

문자열은 프로그램에서 **텍스트 정보를 표현하는 자료형**이다.

숫자가 계산을 담당한다면  
문자열은 **상태, 메시지, 데이터를 설명하는 역할**을 한다.

---

## 문자열이 사용되는 곳

### 1. 로그 출력

프로그램 상태를 설명할 때 사용된다.

예)

[Robot] Moving to target  
[Robot] Battery low  
[Robot] Target reached

로봇 시스템에서는 디버깅과 상태 확인을 위해 로그가 매우 중요하다.

---

### 2. 상태 메시지 표현

프로그램의 현재 상태를 사용자에게 전달한다.

예)

print("Robot started")
print("Robot charging")

---

### 3. 데이터 표현

문자열은 데이터 자체를 표현하기도 한다.

예)

"20260310Sunny"

슬라이싱을 이용하면

date = text[:8]
weather = text[8:]

처럼 데이터를 분리할 수 있다.

---

### 4. 문자열 포매팅

문자열 안에 변수 값을 삽입할 수 있다.

가장 많이 사용하는 방식은 **f-string**이다.

distance = 1.23
print(f"[Robot] distance: {distance}")

로봇 로그나 서버 로그에서 매우 많이 사용된다.

---

## 문자열과 숫자의 관계

프로그램에서는 보통 다음과 같은 구조가 된다.

숫자 → 계산
문자열 → 설명

예)

distance = 2.3
print(f"[Robot] distance: {distance}")

---

## 정리

문자열은 프로그램에서

- 로그 출력
- 상태 메시지
- 데이터 표현

과 같은 **정보 전달 역할**을 한다.

특히 자동화 시스템이나 로봇 프로그램에서는  
**로그 출력과 상태 설명에 매우 중요하다.**
