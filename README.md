# jump-to-python

이 저장소는 로보틱스 소프트웨어 개발 역량을 갖추기 위한
파이썬 기초 학습 기록입니다.

단순히 문법을 암기하는 것이 아니라,
각 개념이 왜 필요한지 이해하고
실제 자동화·로봇 시스템과 어떻게 연결되는지를 고민하며 학습합니다.

---

## 📌 학습 목표

- 파이썬 문법의 구조적 이해
- 문제 해결 중심의 사고 훈련
- 기초 문법을 실전 예제로 확장
- AI → ROS → DevOps로 이어지는 기반 구축

---

## 🧠 학습 원칙

1. 교재 예제를 단순히 따라치지 않는다.
2. 배운 개념을 반드시 변형·확장해본다.
3. 작은 기능이라도 “시스템 관점”에서 생각한다.
4. 학습 과정을 기록하여 성장 흐름을 남긴다.
